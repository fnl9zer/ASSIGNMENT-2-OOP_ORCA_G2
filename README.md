# Space Shooter by Orca (G2)

A 2D arcade space shooter built with C# Windows Forms.

## Features
- Player movement and shooting
- Enemy waves with increasing difficulty
- Obstacle (basic and flaming meteors)
- Health and lives system (3 hearts)
- Score and level tracking
- Pause and replay functionality

## Built With
- C# (.NET Framework)
- Windows Forms (PictureBox, Timer)
- Windows Media Player (WMPLib)

## How to Play
- Arrow keys to move
- Auto-shooting enabled
- Space to pause/unpause
